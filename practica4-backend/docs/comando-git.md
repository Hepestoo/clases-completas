# Comandos GitHub

# Iniciar Repositorio
```
git init
```
- Referencia del repositorio local con el repositorio remoto (GitHub)
````
git remote add origin https://github.com/Hepestoo/clases-completas.git
````
# Validar referencia
````
git remote -v
````
----
````
git add.
git commit - m <proyecto>
git put origin master
````
````
- Para forzar
git push -f origin master
````
